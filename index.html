Cписок команд, які ми розглядали на роботі з терміналом

  1. git clone - Клонує репозиторій

  2. git push - Записує оновлення коду на віддалений репозиторій

  3. git status - показує статус в локальному репозитрії

  4. git add . - гіт починає відслідковувати зміни що внесені в файл

  5. git commit -m “commit text” - зберігає поточний стан файла
